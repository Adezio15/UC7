    Grade Curricular - Desenvolvimento de Sistemas

    Disciplina Carga Horária Resumo Conteúdos Ministrados
    Lógica de Programação 80h Introdução ao pensamento lógico e resolução de problemas com algoritmos. Variáveis, operadores, estruturas condicionais e loops.
    Algoritmos 60h Desenvolvimento de algoritmos para resolver problemas computacionais. Fluxogramas, pseudocódigo e lógica computacional.
    HTML e CSS 70h Criação de páginas web estruturadas e estilizadas. Tags HTML, estrutura de páginas, CSS, layout e responsividade.
    JavaScript 80h Programação para tornar páginas web interativas. Eventos, funções, variáveis, manipulação do DOM.
    Banco de Dados 60h Armazenamento e organização de dados em sistemas. Modelagem de dados, SQL, consultas, tabelas e relacionamentos.
